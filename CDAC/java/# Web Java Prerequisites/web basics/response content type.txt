Content Type

Content Type is also known as MIME (Multipurpose internet Mail Extension) Type. It is a HTTP header that provides the description about what are you (web server) sending to the browser.

MIME is an internet standard that is used for extending the limited capabilities of email by allowing the insertion of sounds, images and text in a message.

Reference - https://developer.mozilla.org/en-US/docs/Web/HTTP/Guides/MIME_types
